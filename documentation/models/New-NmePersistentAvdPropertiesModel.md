# New-NmePersistentAvdPropertiesModel

Create an in-memory object for PersistentAvdProperties.

**Output Type:** `PersistentAvdProperties`

## Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| StartVMOnConnect | Boolean | Yes |  |

## Usage

```powershell
$model = New-NmePersistentAvdPropertiesModel `
    -StartVMOnConnect $true
```
