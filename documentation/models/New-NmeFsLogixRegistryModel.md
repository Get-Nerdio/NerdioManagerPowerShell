# New-NmeFsLogixRegistryModel

Create an in-memory object for FsLogixRegistry.

**Output Type:** `FsLogixRegistry`

## Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| Locations | Object | Yes |  |
| Options | String | Yes |  |

## Usage

```powershell
$model = New-NmeFsLogixRegistryModel `
    -Locations "<Locations>" `
    -Options "<Options>"
```
