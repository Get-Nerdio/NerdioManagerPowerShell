# New-NmeConsoleConnectConnectionModel

Create an in-memory object for ConsoleConnectConnection.

**Output Type:** `ConsoleConnectConnection`

## Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| UserName | String | No |  |

## Usage

```powershell
$model = New-NmeConsoleConnectConnectionModel `
    -UserName "<UserName>"
```
